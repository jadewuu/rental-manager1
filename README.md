# 出租屋管理系统

纯前端、移动优先的出租屋管理 Web 应用，数据存储在浏览器本地，无需服务器，即开即用。

## 功能

- **房间管理** — 添加、编辑、删除房间，记录楼层、面积等信息
- **租客管理** — 登记租客、分配房间、退租处理，查看在租/历史租客
- **抄表记录** — 记录每个房间的水电表读数，自动关联上次读数做参考
- **账单生成** — 根据抄表读数和费率自动计算水费、电费，生成完整账单
- **账单管理** — 查看账单明细、标记已缴、筛选未缴/已缴账单
- **数据导入导出** — 支持导出 JSON 备份和导入恢复

## 技术栈

- Vue 3 (Composition API) — UI 框架
- Tailwind CSS — 样式（移动优先，响应式设计）
- LocalStorage — 数据持久化
- 纯前端，无需后端服务器

## 快速开始

直接在浏览器打开 `index.html` 即可使用。

### 方式一：本地使用

```bash
git clone <仓库地址>
cd rental-manager
open index.html
```

### 方式二：部署到 GitHub Pages

1. Fork / 克隆本仓库
2. 进入仓库 Settings → Pages
3. 选择 `main` 分支，根目录
4. 访问 `https://<你的用户名>.github.io/rental-manager/`

## 使用说明

1. **先添加房间** → 在「房间管理」中添加所有出租房间
2. **登记租客** → 在「租客管理」中为房间分配租客
3. **录入读数** → 每月抄表后在「抄表记录」中录入水电读数
4. **生成账单** → 在「账单管理」中选择房间和月份，自动计算费用
5. **定期导出备份** → 在「设置」中导出数据以防丢失

## 费率设置

在「设置」页面可以修改水费、电费的单价，生成账单时会自动使用设置的费率计算。

## 数据安全

所有数据存储在浏览器的 LocalStorage 中，清除浏览器缓存会导致数据丢失，建议定期导出备份。
